# Crypto Monopoly Game
Crypto Monopoly Game (Smart Contracts)
- ERC20:  Game Token
- ERC721: Property
- Solidity: ^0.6.6
- Tool:  ethereum-waffle
- UnitTest: TypeScript + mocha + chai

# Local Development

The following assumes the use of `node@>=10`.

## Install Dependencies

`yarn`

## Compile Contracts

`yarn compile`

## Run Tests

`yarn test`
